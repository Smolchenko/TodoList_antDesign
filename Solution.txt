To build a simple to-do list application using React and local storage, 
we can follow these steps:

1. Create a new React project using create-react-app command.
2. Install the uuid package to generate unique IDs for each to-do item.
3. Create a TodoList component that displays a list of to-do items.
4. Create a TodoItem component that displays each individual to-do item.
5. Create a TodoForm component that allows users to add new to-do items.
6. Implement local storage to persist the to-do list.
